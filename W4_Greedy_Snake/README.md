<h1><center> 项目名称为： GreedySnakeAutoPathFinding</h1>

## Goal
  
==编写贪吃蛇自动吃食物程序，不限制方法策略, 也不限制寻路算法, 并且不能将自己绕死。==



## Requirements
请见``requirements.txt``
## Usage

  - 1.将文件下载后，点击运行``run.bat``，配置好环境。
  - 2.打开``main.py``，运行即可
## How It Works
  - 1.首先利用``turtle``库绘制贪吃蛇游戏。
  - 2.利用``bfs``算法，寻找贪吃蛇到食物的最短路径。
  - 3.实现贪吃蛇自动寻路吃食物。
