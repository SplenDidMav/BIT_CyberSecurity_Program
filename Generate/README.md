<h1 align="center">
VigenereCipherGenerate
</h1>

> 通过输入明文和密钥，生成密文

待加密文档
-------------
```
plaintext.txt
```

程序
-------------

  - ``VigenereCipher.py  ``    
  #产生密文  


使用
-------------
```
1.双击run.bat  
在命令行状态下输入pip install -r requirements.txt  
2.将待加密密文放入plaintext.txt中
3.运行VigenereCipher.py  
4.输入密钥
5.密文将会存放在ciphertext.txt中
```
可能存在的bug
-------------
```
密文输出没有空格
```
